# Mediawise Solutions Logo Usage Guide

## Logo Overview

The Mediawise Solutions logo features a modern, geometric chevron mark combined with clean typography. The design communicates growth, direction, and professionalism—key attributes of the brand.

**Mark Design:**
- Three-tier stacked chevron
- Primary Teal (#3da89a)
- Secondary Gold (#d4a574) bottom tier
- Bold, geometric, distinctive

---

## Logo Versions

### 1. Primary Dark (Use on Dark Backgrounds)

**File:** `Primary Dark/1_Vertical.png` and `2_Horizontal.png`

**Use When:**
- Dark navy or teal backgrounds
- Website headers on dark backgrounds
- Printed materials with dark colors
- Design layouts with dark primary color

**Color:** Full color (Teal + Gold)

**Examples:**
- Website header on dark background
- Email signatures with dark background
- Printed letterhead on colored stock
- Social media headers with dark theme

---

### 2. Secondary Light (Use on Light Backgrounds)

**File:** `Secondary Light/1_Vertical.png` and `2_Horizontal.png`

**Use When:**
- White or light backgrounds
- Light-colored paper or materials
- Reversed/inverted applications
- Maximum contrast needed

**Color:** Full color (Teal + Gold) on light background

**Examples:**
- Standard website/app logos
- Printed materials on white paper
- Light colored backgrounds
- Standard business applications

---

## Layout Orientations

### Vertical Layout
**Best For:**
- Square spaces (social media profiles)
- Headers with vertical emphasis
- Stacked layouts
- Small spaces requiring clarity

**Usage:** Use when height is equal to or greater than width

### Horizontal Layout
**Best For:**
- Wide formats (email headers, website banners)
- Landscape-oriented spaces
- Horizontal navigation bars
- Wide layouts with limited height

**Usage:** Use when width is significantly greater than height

---

## Clear Space Requirements

**Minimum Clear Space:** 20px on all sides (or 1/4 of logo height, whichever is greater)

This means:
- 20 pixels of empty space around the entire logo
- No text, images, or design elements within this space
- Applies regardless of logo size or orientation

**Example:**
```
[20px]  [ LOGO ]  [20px]
[20px]  [      ]  [20px]
```

---

## Size Requirements

### Digital Usage
- **Minimum Size:** 64px height
- **Recommended Sizes:** 128px, 256px, 512px (powers of 2 for web)
- **Maximum Size:** No limit (maintain aspect ratio)

### Print Usage
- **Minimum Size:** 1 inch (25.4mm) height
- **Recommended Sizes:** 1", 1.5", 2"
- **Resolution:** 300 DPI minimum

### Logo File Formats
- **PNG (Transparent):** Primary format for digital
- **PDF:** For print and scalable use
- **SVG:** For web and responsive scaling (if available)

---

## Color Specifications

### Teal Primary (#02d8be)
- **Hex:** #02d8be
- **RGB:** 2, 216, 190
- **CMYK:** 99, 5, 30, 0 (approximate for print)
- **Usage:** Main chevron color, primary brand applications

### Gold Secondary (#d8c38a)
- **Hex:** #d8c38a
- **RGB:** 216, 195, 138
- **CMYK:** 0, 10, 36, 15 (approximate for print)
- **Usage:** Bottom chevron tier, secondary accent

**Important:** Always use these exact colors. Never alter, desaturate, or change the logo colors.

---

## Do's for Logo Usage

✓ **DO:**
- Use official logo files only
- Maintain aspect ratio when resizing
- Use correct version for background (dark or light)
- Keep clear space around logo
- Ensure sufficient size for clarity
- Test on actual background before finalizing
- Use high-resolution versions for print
- Position logo consistently across materials

---

## Don'ts for Logo Usage

✗ **DON'T:**
- Alter logo colors or proportions
- Stretch or distort the logo
- Add effects (shadows, glows, gradients)
- Place on competing patterns or images
- Use wrong version for background
- Reduce size below minimum requirements
- Rotate logo at odd angles
- Recreate logo (always use official files)
- Use low-resolution versions for print
- Place in cluttered layouts

## Print Implementation

### Business Cards
- Vertical logo top-left or bottom-right
- Size: 0.75" height
- 300 DPI minimum resolution

### Letterhead
- Horizontal logo, top center or top-left
- Size: 1.5" width
- 300 DPI minimum resolution

### Printed Marketing Materials
- Vertical or horizontal depending on layout
- Minimum 1" height
- 300 DPI resolution
- Use CMYK color space for printing
