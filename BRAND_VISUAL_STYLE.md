## Typography System

### Heading Styles

**Headline Level 1 (H1)**
- Font: Bold Sans-Serif
- Size: 48pt or larger
- Case: UPPERCASE
- Color: White (#ffffff)
- Line-height: 1.2
- Example usage: Main page titles, hero section

**Headline Level 2 (H2)**
- Font: Bold Sans-Serif
- Size: 36pt
- Case: UPPERCASE
- Color: White (#ffffff)
- Line-height: 1.2
- Example usage: Section headings

**Headline Level 3 (H3)**
- Font: Bold Sans-Serif
- Size: 24pt
- Case: UPPERCASE or Title Case
- Color: White (#ffffff)
- Example usage: Subsection titles, card titles

**Body Text**
- Font: Regular Sans-Serif
- Size: 16pt (desktop), 14pt (mobile)
- Color: Light Gray (#a8b8bb)
- Line-height: 1.6
- Max width: 600px for readability

**Accent Text**
- Style: White text with Gold underline
- Usage: Emphasizing key words or phrases within body text

## Component Styles

### Call-to-Action Buttons

**Primary CTA:**
- Background: Teal Accent (#3da89a)
- Text: White, bold, uppercase
- Padding: 12px 24px
- Border-radius: 0px (sharp corners)
- Hover: Slightly darker teal or white text
- Example: "WORK WITH ME", "START A CONVERSATION"

**Secondary CTA:**
- Background: Transparent
- Border: 1px Gold Accent (#d4a574)
- Text: Gold or White
- Padding: 12px 24px
- Hover: Gold background

### Cards & Boxes

- Background: Slightly lighter than primary background (or transparent with border)
- Border: 1px Gold or Teal
- Padding: 24px
- Shadow: Minimal (no drop shadows)
- Corners: Sharp (0px border-radius)

### Icons & Visual Accents

- Color: Teal Accent primary, Gold for secondary
- Style: Clean, minimal, modern
- Size: Maintain 1:1 aspect ratio
- Usage: Next to headings, in lists, highlighting key points

### Whitespace
- Generous whitespace around content

2. **H2** — Section importance (section headings)

5. **Accent** — Emphasis on key concepts

## Responsive Design

### Desktop (1200px+)
- Full-width layouts with generous margins
- Large typography
- Multi-column layouts
- Full navigation visible

### Tablet (768px - 1199px)
- Adjusted margins and padding
- Slightly reduced typography
- Two-column layouts where appropriate
- Optimized navigation

### Mobile (< 768px)
- Single column layouts
- Stacked content
- Reduced typography while maintaining hierarchy
- Mobile-friendly navigation
- Touch-optimized buttons (minimum 44px height)

## Logo Usage Rules

### When to Use Each Version

**Vertical (Dark Background):** Primary logo on dark teal/navy backgrounds  
- Website header, printed materials, dark design backgrounds

**Vertical (Light Background):** Logo on light/white backgrounds  
- Light colored papers, light backgrounds, reversed logos

**Horizontal (Dark Background):** Landscape format on dark backgrounds  
- Email headers, social media covers, wide format applications

**Horizontal (Light Background):** Landscape format on light backgrounds  
- Light colored backgrounds, horizontal layouts

### Clear Space & Placement

- Maintain clear space around logo (minimum 20px on all sides)
- Never place logo on competing patterns or images
- Use logo as anchor for design (typically top-left or center)
- Ensure logo is never smaller than 64px height (digital)

### Color Variants

- **Full Color (Primary):** Teal chevron with gold bottom tier
- **Single Color (White):** Use on dark backgrounds only
- **Single Color (Dark):** Use on light backgrounds only
- Never change logo colors (maintain teal and gold)

## Do's & Don'ts for Visual Design

### ✓ DO:
- Use dark backgrounds for authority and contrast
- Employ bold, large typography
- Apply teal and gold strategically for accent
- Maintain generous whitespace
- Keep designs clean and minimal
- Use professional, authentic imagery
- Ensure high text contrast
- Test across all device sizes
- Use official logo files in correct color versions
- Maintain proper clear space around logo

### ✗ DON'T:
- Use bright or distracting colors
- Add unnecessary decoration or patterns
- Use small, hard-to-read fonts
- Crowd content or reduce whitespace
- Mix inconsistent design styles
- Use generic stock photos
- Apply drop shadows or 3D effects
- Over-animate or distract from content
- Alter logo colors, proportions, or style
- Place logo on competing backgrounds without contrast

## Accessibility & Contrast

### Color Contrast Ratios
- All text must meet WCAG AA standards (4.5:1 minimum)
- Dark Teal background + White text = excellent contrast
- Light Gray text on Dark Teal = good contrast
- Test all color combinations for accessibility

### Font Sizing
- Minimum 16px for body text
- 1.5-2x multiplier for accessible heading size
- Adequate line-height (1.5-1.6 minimum)
- Avoid all-caps for large body text blocks

### Interactive Elements
- Buttons: minimum 44px height/width for touch
- Links: underlined or clearly distinguishable
- Focus states visible for keyboard navigation
- Color not sole indicator of state/meaning

## File Formats & Specifications

### Web Usage
- PNG for logos and graphics (transparent background)
- JPG for photographs
- SVG for icons (scalable)
- WebP for modern browser optimization

### Print Usage
- High-resolution PDFs
- CMYK color mode (not RGB)
- 300 DPI minimum
- Include bleed area (0.125")

### Digital Assets
- Maintain source files in multiple formats
- Document color values (Hex and RGB)
- Keep typography specifications documented
- Archive all brand materials
