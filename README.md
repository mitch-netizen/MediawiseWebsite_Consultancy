# Mediawise Solutions Website

Portfolio and consultancy website for Mitch Harris - Marketing & Revenue Strategist.

## About

A static HTML website showcasing marketing consultancy services based in Central Queensland, Australia.

## Deployment

Deployed on Vercel at [mediawisesolutions.com.au](https://mediawisesolutions.com.au)

### Local Development

Since this is a static site, no build process is required. Simply open `index.html` in a browser to preview locally.

### Deploying to Vercel

The site is automatically deployed to Vercel whenever changes are pushed to the main branch.

```bash
git add .
git commit -m "Update website"
git push origin main
```

## Project Structure

- `index.html` - Main website file (all content in one file)
- `vercel.json` - Vercel configuration
- `.gitignore` - Git ignore rules
